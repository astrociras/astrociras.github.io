# CIRAS-AI website

Static bilingual website for CIRAS-AI.

- `/index.html`: Spanish landing page
- `/en/index.html`: English landing page
- `/legacy/`: archived CIRAS website
- `/assets/css/ciras-ai.css`: project-specific styles
- `/images/`: optimized public images and logos

Serve locally from this directory with:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/`, `/en/`, and `/legacy/`.
